## 项目概述

本项目是基于Python的智慧小区物业管理系统，使用Django框架和Vue.js进行前后端开发，使用MySQL作为数据库管理系统。

## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL
